# BudgetCentre macOS App

BudgetCentre for macOS is a secure Electron container for `https://bc.tool.axchen.top`.
It requires an internet connection and keeps its browser session separate from regular browsers.

## Commands

```bash
yarn
yarn run check
yarn test
yarn package:mac-arm64
```

The Apple Silicon DMG is written to `release/`. It is intentionally unsigned for this first release.
